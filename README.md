# Stumpy the Link Shortener

Stumpy is a link shortener written with Meteor for the [_EANTE Internship Challenge_](https://github.com/Eante/Internship-Challenge-2017).

### features

 - Make links shorter
 - Login to use custom short URLs and view all links
 - Track link clicks
 - looks ok
 - doesn't work with javascript disabled
